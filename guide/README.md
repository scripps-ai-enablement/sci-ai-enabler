# Guide

Short, beginner-facing pages that explain how Claude's component model works — Skills, MCP servers, Plugins, Marketplaces, Connectors — and how to install them. Each page is ≤ 400 words and links to canonical Anthropic docs rather than reproducing them.

For lists of specific life-science tools that fit each component type, see the [catalog](../catalog/).

## Reading order

1. [Claude surfaces](claude-surfaces.md) — Claude.ai, Claude Desktop, Claude Code, and the Claude API. Start here if you're not yet sure which one you're using.
2. [Skills](skills.md) — focused capabilities a Claude agent can adopt.
3. [MCP servers](mcp-servers.md) — external tools and data sources Claude can call.
4. [Plugins](plugins.md) — bundles that combine skills, MCP servers, and hooks.
5. [Marketplaces](marketplaces.md) — where to find plugins and skills, and how to add one to Claude Code.
6. [Connectors](connectors.md) — Anthropic-managed integrations available inside Claude.ai.
7. [Decision tree](decision-tree.md) — "I want to do X — which of the above should I use?"

## Advanced

When you're comfortable with the basics, the [advanced section](advanced/) covers hooks, custom slash commands and subagents, and authentication.
