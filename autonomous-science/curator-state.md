# Curator state

## Recently surfaced

- **EvoMaster** (added 2026-05-27) — SJTU / SciLand / DP Technology foundational ~100-line harness for building self-evolving scientific agents; powers the SciMaster ecosystem (ML-Master, X-Master, Browse-Master, PhysMaster, EmboMaster); SOTA on four benchmarks with GPT-5.4 (HLE 41.1%, MLE-Bench Lite 75.8%, BrowseComp 73.3%, FrontierScience 53.3%) at +159% to +316% over OpenClaw; open source.
- **NeuroClaw** (added 2026-05-27) — CUHK / Northwest U / Lehigh / MGH / Harvard Kempner multi-agent assistant for executable and reproducible neuroimaging research operating on raw sMRI/fMRI/dMRI/EEG; three-tier skill hierarchy + harness with pinned Python/Docker and BIDS-aware orchestration; companion NeuroBench (100 tasks) shows all ten frontier multimodal LLMs improve under NeuroClaw (mean +4.74 pts; Claude-Opus-4.6 72.10%); adds neuroimaging as a new domain.
- **Aleks** (added 2026-05-25) — Cornell three-agent system (Domain Scientist, Data Analyst, ML Engineer) that autonomously frames a problem, iterates ML strategies, and produces interpretable models from a research question + dataset; grapevine red blotch disease case study with ablations showing domain knowledge and cross-iteration memory are both load-bearing.
- **Deep Researcher Agent** (added 2026-05-25) — University of Tokyo open-source 24/7 Think→Execute→Reflect framework for autonomous deep-learning experimentation; Zero-Cost Monitoring keeps LLM cost at $0.08/24h cycle; 500+ cycles and 52% best-project improvement over 30+ days; open source.
- **PerTurboAgent** (added 2026-05-25) — Genentech/Regev-lab LLM agent for iterative Perturb-seq experiment design with an Action Pool (predict/reflect/refine + data-driven GSEA + model-driven prediction) and Action Memory; outperforms BioDiscoveryAgent, GeneDisco, DiscoBAX, and IterPert across eleven phenotypes.
