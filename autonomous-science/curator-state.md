# Curator state

## Recently surfaced

- **Latent-Y** (added 2026-05-29) — Latent Labs lab-validated autonomous agent for de novo biologics design that runs literature review, target/epitope analysis, candidate design via Latent-X2, computational validation, and lab-ready selection from a text prompt; 67% target-level success rate (6/9), single-digit nanomolar VHH affinities (5.44 nM PRL, 12.5 nM IL-6), and 56× expert-campaign acceleration.
- **PantheonOS** (added 2026-05-29) — Stanford (Qiu lab) evolvable multi-agent genomics framework spanning RL gene-panel design, FASTQ processing, multi-omics integration, and 3D spatial reconstruction; Pantheon-Evolve performs agentic code evolution that rewrites batch-correction and gene-panel algorithms beyond manual baselines; demonstrated discoveries include asymmetric Cer1-Nodal inhibition in mouse E6.0 and fetal-heart multi-omics integration.
- **The Virtual Biotech** (added 2026-05-29) — Stanford (Zou lab) multi-agent framework mirroring a biotech company structure (CSO agent + domain scientist agents in statistical genetics, functional genomics, chemoinformatics, clinical data); >37,000 clinical-trialist agents analyzed 55,984 trials and found drugs targeting cell-type-specific genes are 40%/48% more likely to advance Phase I→II / reach Phase IV with 32% lower AEs.
- **EvoMaster** (added 2026-05-27) — SJTU / SciLand / DP Technology foundational ~100-line harness for building self-evolving scientific agents; powers the SciMaster ecosystem (ML-Master, X-Master, Browse-Master, PhysMaster, EmboMaster); SOTA on four benchmarks with GPT-5.4 (HLE 41.1%, MLE-Bench Lite 75.8%, BrowseComp 73.3%, FrontierScience 53.3%) at +159% to +316% over OpenClaw; open source.
- **NeuroClaw** (added 2026-05-27) — CUHK / Northwest U / Lehigh / MGH / Harvard Kempner multi-agent assistant for executable and reproducible neuroimaging research operating on raw sMRI/fMRI/dMRI/EEG; three-tier skill hierarchy + harness with pinned Python/Docker and BIDS-aware orchestration; companion NeuroBench (100 tasks) shows all ten frontier multimodal LLMs improve under NeuroClaw (mean +4.74 pts; Claude-Opus-4.6 72.10%); adds neuroimaging as a new domain.
